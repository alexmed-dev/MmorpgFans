Разработан интернет-ресурс "Моя Игромания" для фанатского сервера одной известной MMORPG.

1. Доступ.

    Доступ на сайт открыт для всех.

    Любой пользователь может:
    1. — Просматривать список обявлений.
    2. — Просматривать любое объявление и отклики к нему, которые приняты автором объявления.
    3. — Зарегистироваться на сайте.
    Для остальных действие необходимо зарегистрироваться и зайти на сайт.

    Авторизованный пользователь может:
    1. — Просматривать список объявлений.
    2. — Просматривать любое объявление и отклики к нему, которые приняты автором объявления.
    3. — Добавить отклик на любое, кроме своего, объявление.
    4. — Добавить свое объявление.
    5. — Редактировать свое объявление.
    6. — На странице "Профиль":
        - просматривать все отклики на свои объявления;
        - фильтровать отклики по названию объявления, по содержанию отклика;
        - принять отклик, после чего отклик будет виден всем посетителям сайта;
        - удалить отклик на свое объявление.


2. Основные действия.

    Регистрация.
    При регистрации надо указать e-mail. На него будет отправлена ссылка для подтверждения регистрации.

    Добавление объявления.
    Доступно авторизованному пользователю.

    Добавление отклика.
    Доступно авторизованному пользователю.
    На любое объявление, кроме своего.
    После сохранения отклика, автору объявления отправляется письмо на e-mail с информацией о добавленном отклике, названии объявления и ссылка на страницу профиля, отфильтрованной по этому объявлению.

    Принять отклик.
    Доступно автору объявления, на которые отправлен отклик.
    После принятия отклика, его автору отправляется письмо на e-mail с информацией о том, что отклик принят.

    Удалить отклик.
    Доступно автору объявления, на которое отправлен отклик.


3. Страницы.

    - Объявления
      Список всех обявлений. Доступен всем.
      Информация:
      - заголовок объявления;
      - Превью - первые 123 символа (удаляются изображения);
      - дата создания и автор;
      - только для автора объявления - ссылка "редактировать".
    - Добавить объявление. Только для авторизованных пользователей.
    - Объявление - просмотр. Доступен всем.
      Информация:
      - заголовок объявления - в шапке;
      - дата создания и автор - в шапке;
      - полное содержание, включая текст, изображения, ссылки и др.;
      - список откликов на данное объявление - всем видны только принятые новости;
      - только для автора объявления - ссылка "редактировать";
      - для авторизованных пользователей, кроме автора: Добавить отклик;
    - Объявление - редактировать. Только для автора объявления.
    - Профиль. Только для авторизованных пользователей
      - список всех откликов на свои объявления;
      - ссылки на принять отклик (если не принято)/ информация, что отклик принят;
      - ссылки на удаление отклика.
    - Вход/Выход

4. Почта

Любой посетитель сайта может зарегистрироваться на странице регисрации, указав свой e-mail и пароль для сайта.
После регистрации на указанный e-mail придет письмо с ссылкой для подверждения регистрации.

Любой пользователь (кто прошел регистрацию) получает почтовые рассылки о новых объявлениях за прошедшую неделю.

Любой пользователь (кто прошел регистрацию) получает информацию о добавлении отклика на свое объявление.

Любой пользователь (кто прошел регистрацию) получает информацию о принятии своего отклика автором объявления.
